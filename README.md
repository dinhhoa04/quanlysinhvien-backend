# Student Management System

A comprehensive academic portal facilitating automated enrollment and role-based grade management.

## 📸 Screenshots

## 🚀 Key Features
- **RBAC & Security:** Role-Based Access Control (Admin, Teacher, Student) implemented with JWT and secure BCrypt password encryption.
- **Academic Management:** Centralized control for Departments, Subjects, Classes, and Teachers; features automated student enrollment, real-time grading, and dynamic GPA calculation.
- **User Profile:** Self-service profile editing and secure password update functionality for all user roles.

## 🛠️ Tech Stack
- **Backend:** Java, Spring Boot, Spring Security, JWT, MySQL
- **Frontend:** ReactJS, Bootstrap 5, Axios
- **DevOps & Tools:** Docker, Postman, Git

## 💻 How to Run Locally
1. Clone both backend and frontend repositories to your local machine
2. Configure your MySQL database connection details in the backend application properties
3. Use Docker to containerize and spin up the environments, or run the backend via Maven and the frontend via Node.js directly.
